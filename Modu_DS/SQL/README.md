# SQL 학습
이 저장소는 SQL 기초 문법을 학습하며 정리한 Jupyter Notebook 모음입니다.     
다양한 실습용 CSV 파일과 함께 SELECT 문, JOIN, GROUP BY, 서브쿼리 등 주요 문법들을 단계별로 정리하였습니다.    

## 디렉토리 구조
```phsql
sql/
│
├── data/                # 실습용 CSV 파일 (customers, employees, orders 등)
│   ├── bike_dataset.csv
│   ├── customers.csv
│   ├── DS_JobsSalaries_2020.csv
│   ├── DS_JobsSalaries_2021.csv
│   ├── employees.csv
│   └── orders.csv
│
├── images/              # 시각화나 결과 캡처 이미지 저장용 폴더
│
├── SQL 정리노트_1.ipynb  # SQL 기초 개념
├── SQL 정리노트_2.ipynb  # SELECT ~ FROM ~ WHERE 문법
├── SQL 정리노트_3.ipynb  # JOIN과 서브쿼리 실습
└── SQL 정리노트_4.ipynb  # GROUP BY 및 집계 함수 실습
```

## 노트북 요약
| 파일명                   | 요약                                                              |
| --------------------- | --------------------------------------------------------------- |

| **SQL 정리노트\_4.ipynb** | 데이터를 그룹화하고, 집계함수를 사용해 간단한 통계를 계산하는 방법                           |
