# 혼공머신 학습 정리

『혼자 공부하는 머신러닝+딥러닝』(한빛미디어)을 읽고 학습한 내용을 정리한 저장소입니다.

## 정리 방식

- 책의 챕터 순서를 그대로 따라가기보다는 주요 모델별로 정리했습니다.
- 각 모델을 학습하며 책에서 다룬 핵심 개념과 이론도 함께 정리했습니다.
- 실습 예제는 책의 예제를 그대로 사용하지 않고 직접 구성한 데이터와 코드로 새롭게 작성했습니다. (예: 도미/빙어 데이터 대신 iris, wine, diabetes 등 사이킷런 공개 데이터셋 활용)
- 단순히 코드를 따라 치는 것이 아니라 개념을 응용해서 직접 실험하고 그 결과를 해석하는 방식으로 정리했습니다. (예: 하이퍼파라미터 변화에 따른 성능 비교, 표준화 전/후 비교, 모델 한계 직접 검증 등)

## ⚠️ 참고

본 저장소는 책 내용을 그대로 옮긴 것이 아니라, 학습한 개념을 직접 이해하고 재구성한 개인 학습 기록입니다.
